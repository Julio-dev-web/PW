O problema no seu código está na linha:

JavaScript

nome.innerHTML = `${nome}`;
Você está tentando usar innerHTML em uma variável que armazena o valor do campo de texto (nome), e não no elemento HTML onde você deseja exibir o nome.

Aqui está o código corrigido:

HTML

<!DOCTYPE html>
<html>
<head>
    <title>Mostrar Nome</title>
</head>
<body>
    Nome:<input type="text" name="" id="nome">
    <button onclick="Mostrar()">submit</button>

    <br><br>

    <div id="meunome">
        Digite o seu nome
    </div>

    <script>
        function Mostrar() {
            let show = document.getElementById('meunome');
            let nome = document.getElementById('nome').value;

            show.innerHTML = `${nome}`; // Correção: usa innerHTML no elemento correto
        }
    </script>
</body>
</html>
Explicação da Correção:

show.innerHTML =${nome};:
Em vez de tentar usar innerHTML na variável nome (que contém apenas o texto digitado), usamos innerHTML no elemento show (que é o div com o id "meunome").
Isso define o conteúdo HTML do div "meunome" para o valor digitado no campo de texto.
Como o Código Corrigido Funciona:

O usuário digita um nome no campo de texto com id="nome".
Quando o botão "submit" é clicado, a função Mostrar() é chamada.
Dentro da função Mostrar():
document.getElementById('meunome') obtém o elemento div com o id "meunome".
document.getElementById('nome').value obtém o valor digitado no campo de texto.
show.innerHTML =${nome}; define o conteúdo HTML do div "meunome" para o valor digitado.
O nome digitado é exibido dentro do div "meunome".
